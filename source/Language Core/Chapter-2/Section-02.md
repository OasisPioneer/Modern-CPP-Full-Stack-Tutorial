# 基本数据类型

数据类型（Data Type）是⼀种属性，它告诉编译器如何解释存储在计算机内存中的某段数据。数据类型决定了数据的存储⼤⼩、布局、值范围以及可以对该数据执⾏哪些操作。

C++ 提供了多种数据类型，包括基本数据类型、复合数据类型和特殊数据类型，本篇主要讲解的是 C++ 的基本数据类型。

基本数据类型分为`整数型`、`浮点型`、`字符型`、`布尔型`、`空类型`。它们在各自对应的应用场景中都有着不可或缺的重要性，接下来对这些数据类型进行逐步的学习。

## 整数型

整数型可以理解为在内存中开辟一个专门用于存储整数的空间，根据要存放的数字大小可以采用不同的整数型关键字来开辟所需大小的空间。

| 类型        | 占用字节 | 存储范围                                                    |
| ----------- | -------------- | ----------------------------------------------------------- |
| `short`     | 2              | `-32,768` 到 `32,767`                                       |
| `int`       | 4              | `-2,147,483,648` 到 `2,147,483,647`                         |
| `long`      | 4 或 8         | 依赖于系统和编译器                                          |
| `long long` | 8              |`-9,223,372,036,854,775,808` 到 `9,223,372,036,854,775,807`|

```CPP
#include <iostream>

int main()
{
    short S = 32767;
    int I = 2147483647;
    long L = 2147483647;
    long long LL = 9223372036854775807;

    std::cout << "短整型: " << S << std::endl;
    std::cout << "整型: " << I << std::endl;
    std::cout << "长整型: " << L << std::endl;
    std::cout << "超长整型: " << LL << std::endl;

    return 0;
}
```

## 浮点型

对于一些需要存储小数的场景下我们需要使用浮点数据类型来存储，例如: `3.14`

| 类型          | 占用大小 | 范围                                                 |
| ------------- | -------------------- | ---------------------------------------------------- |
| `float`       | 4                    | 约 6-7 位十进制小数精度                              |
| `double`      | 8                    | 约 15-16 位十进制小数精度                            |
| `long double` | 8, 12, 或 16         | 提供比 `double` 更高的精度，具体大小和精度依赖于环境 |

```CPP
#include <iostream>

int main() {
    float myFloat = 3.14;
    double myDouble = 3.14159;
    long double myLongDouble = 3.1415926535;
    
    std::cout << "单精度浮点数: " << myFloat << std::endl;
    std::cout << "双精度浮点数: " << myDouble << std::endl;
    std::cout << "长双精度浮点数: " << myLongDouble << std::endl;
    
    return 0;
}

```

```{attention}
浮点类型可以表⽰⾮常⼤或⾮常⼩的数，但它们只能近似地表⽰⼩数，因为它们在内存中的表⽰是基于⼆进制分数的。

浮点类型在计算机中的表示有时会导致出现精度问题，因此在需要进行精确计算的场景（如金融计算）中，应该考虑使用专门的库而不是浮点类型。
```

## 字符型

字符型是一个用于存储各种文本字符的数据类型例如: `1234567890` 或 `a-z` 或 `A-Z`，这些字符都可以使用此数据类型进行存储。

| 类型      | 占用字节 | 范围                                                   |
| --------- | -------------------- | ------------------------------------------------------ |
| `char`    | 1                    | 用于存储单个ASCII字符，或 `-128` 到 `127` 范围的小整数 |

`char` 数据类型通常只能用来存储一个字符 也就是 "1" / "A" 这种单个的字符，如果想要存储一句话这种长字符就需要用到**数组**来实现。

```CPP
#include <iostream>

int main() {
    char character = 'A';
    wchar_t wideChar = L'Ω';
    
    std::cout << "字符: " << character << std::endl;
    std::cout << "字符的ASCII值: " << int(character) << std::endl;
    
    return 0;
}
```

## 布尔型

布尔型也可以称之为逻辑型，这个数据类型只能存储 "`true`" / "`false`" 两个值。布尔型常用于需要进行逻辑判断的场景。

| 类型   | 占用字节 | 范围                            |
| ------ | -------------------- | ------------------------------- |
| `bool` | 1                    | 只能是 `true` 或 `false` 两个值 |

```CPP
#include <iostream>

int main() {
    bool isTrue = true;
    bool isFalse = false;
    
    std::cout << "isTrue: " << isTrue << std::endl;
    std::cout << "isFalse: " << isFalse << std::endl;
    std::cout << "isTrue的实际值: " << (isTrue ? "true" : "false") << std::endl;
    
    return 0;
}
```

## 空类型

空类型代表着没有类型，使用场景通常是表示函数没有返回值或者函数不需要接收参数。

| 类型   | 占用字节 | 范围                                               |
| ------ | -------------------- | -------------------------------------------------- |
| `void` | 不适用               | 不适用 |

```CPP
void demo(void){
    std::cout << "我不接收任何参数 也不返回任何值" << std::endl;
}
```

```{error}
如果某个数据类型存储范围的数据超出其所能够存储的范围则会发生数据溢出。

每种数据类型只能用于存储符合该类型的数据，例如使用整数类型来存储例如: `3.1415` 或 `a` 这种数据则同样会出现错误!!!
```
